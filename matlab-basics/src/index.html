Introduction

MATLABÂ® (Matrix Laboratory) is a programming language designed for numerical computations. It is a proprietary language, meaning that it cannot be freely used outside of the jurisdiction of MathWorks.

Cleve Moler, the chairman of the computer science department at the University of New Mexico, started developing MATLAB in the late 1970s to make mathematical programming easier. It is a popular language in control engineering and image processing, among many other domains. It is influenced by Fortran. There are several additional toolboxes available; for example:

  • The Image Processing Toolbox provides algorithms for image processing, analysis, and visualization. Possible applications include image segmentation, image enhancement, noise reduction, geometric transformations, image registration, and 3D image processing.
  • Simulink is a MATLAB-based graphical programming environment for dynamical systems. The software uses a graphical block diagramming tool and a customizable set of libraries. It is popular in control systems and signal processing for its flowchart capabilities, and can drive MATLAB or be scripted from it.

MATLAB and Simulink

The main products of MathWorks are MATLAB and Simulink. MATLAB is both a programming language and a software application. Code is executed through the command window or through running .m files. MATLAB programs are built primarily using variables, vectors, and matrices.

Simulink has a Graphical User Interface (GUI) whereas Matlab is only code. Simulink's GUI gives one the opportunity to create flowcharts. Simulink can interface with hardware such as robots and motors. It is useful for creating a PID Controller, which creates continuously modulated control.

MATLAB can embed code, meaning that it can convert Simulink code to C code. One limitation of Matlab is that it doesn't control hardware very well, as it does not have the required libraries.

Hello world
To simply print "Hello world", type the following into the MATLAB terminal: disp('Hello world'); This will simply print 'Hello world'. If you wish to create a function, you have to create a file with an input and output variable, then call the function from the MATLAB terminal: 
function f = greetMe(n)
    f = "Hello " + n;
end
greetMe("World");
Variables

You use variables as symbolic names for values in your application. The names of variables must conform to certain rules.

A valid MATLAB variable name starts with a letter, followed by letters, digits, or underscores. MATLAB is case sensitive, so A and a are not the same variable. The maximum length of a variable name is the value that the "namelengthmax" command returns. One cannot define variables with the same names as MATLAB keywords, such as "if" or "end". For a complete list, run the "iskeyword" command.

If you create a variable that uses the name of a function, you sometimes get unexpected results.

Declaring variables
Variables do not have to be declared before assigning values to them.

Enter the variable name in the Command Window, followed by an equal sign (=) and the value you want to assign to the variable. For example: var x = 42.

If you do not end the assignment statement with a semicolon (;), MATLAB displays the result in the Command Window. For example: x = 75 will return x = 75.0000

If you do not assign the output of a statement to a variable, MATLAB typically assigns the result to the reserved word 'ans'. For example,13 will return ans = 13.0000
